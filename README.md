## 🌱 Contribute Steps
- Fork This repo
- Copy repo url which is created after forked 

```bash
$git <repo url>
```

- Add Your Codes

```bash
$git add .
```

```bash
$git commit -m "<message>"
```

```bash
$git push origin main
```

- Then open your forked repo and click on ___contribute___ to create pull request
